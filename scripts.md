# Scripts

```bash
npx typeorm-populate <command>
```

| Option  |               Description                |
| ------- | :--------------------------------------: |
| cli     | an interactive cli that has every option |
| init    |           initialize the files           |
| seed    |        seed your database defined        |
| factory |           generates a factory            |
