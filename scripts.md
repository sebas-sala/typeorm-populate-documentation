# Scripts

```bash
npx typeorm-populate <command>
```

| Option  |                  Description                  |
| ------- | :-------------------------------------------: |
| cli     | Open an interactive cli that has every option |
| init    |             initialize the files              |
| seed    |          seed youre database defined          |
| factoyr |              generates a factory              |
